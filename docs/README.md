# Docs for https-proxy

This `docs` directory contains short guides and design notes for the small `https-proxy` project. The content is aimed at learners who want to understand how a minimal HTTP/HTTPS proxy is implemented using Go's standard library.

Files:

- `usage.md` — examples showing how to run and test the proxy.
- `design.md` — a short code walkthrough and important implementation details.

Read the individual documents for details.
